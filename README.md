# 🍽️ Restaurant Order System

A simple and clean web-based restaurant ordering system built with HTML, CSS, and JavaScript.

## 📁 Project Structure

```
restaurant-order-system/
├── index.html       → Main landing page
├── menu.html        → Food menu page
├── cart.html        → Shopping cart page
├── order.html       → Order confirmation page
├── style.css        → Global styles
├── menu.css         → Menu page styles
├── cart.css         → Cart page styles
├── app.js           → Main app logic
├── cart.js          → Cart functionality
├── order.js         → Order management
└── README.md        → Project documentation
```

## 🚀 Features

- Browse restaurant menu by category
- Add items to cart
- View and edit cart
- Place and confirm orders

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)

## 📦 How to Use

1. Clone the repository
2. Open `index.html` in your browser
3. No build tools or dependencies required
